# 碎石记（Crusher）后台管理使用指导手册

## 一、后台登录
1. 使用管理员账号登录平台
2. 访问`/admin`路径进入管理后台
3. 系统会自动验证用户角色，非管理员账号无法访问后台

## 二、用户管理
### 2.1 用户列表
1. 查看所有注册用户的基本信息：ID、邮箱、用户名、注册时间、当前积分、账号状态
2. 支持按邮箱、用户名搜索用户
3. 支持按注册时间排序

### 2.2 用户操作
1. **调整积分**：手动为用户增加或扣除积分，需填写操作说明
2. **禁用账号**：封禁违规用户，封禁后用户无法登录
3. **角色变更**：将普通用户升级为管理员，或将管理员降级为普通用户
4. **查看详情**：查看用户的所有文档、分析记录、交易记录

## 三、订单管理
### 3.1 订单列表
1. 查看所有支付订单：订单号、用户信息、套餐类型、金额、支付状态、创建时间
2. 支持按订单状态筛选：待支付、已支付、支付失败、已退款
3. 支持按用户、订单号搜索订单

### 3.2 订单操作
1. **手动确认到账**：当支付回调异常时，可手动标记订单为已支付，并为用户发放积分
2. **退款处理**：处理用户退款申请，扣除对应积分后标记订单为已退款
3. **导出订单**：导出指定时间范围的订单数据用于财务对账

## 四、交易记录管理
1. 查看所有用户的积分变动记录：用户信息、变动金额、变动类型（购买、消费、手动调整、退款等）、变动时间、操作人
2. 支持按交易类型、用户筛选记录
3. 支持导出交易记录用于审计

## 五、反馈管理
### 5.1 反馈列表
1. 查看用户提交的所有反馈：类型（bug、功能建议、支付问题、其他）、标题、内容、提交用户、状态、提交时间
2. 支持按反馈类型、状态筛选

### 5.2 反馈处理
1. 标记反馈状态：待处理、处理中、已解决、已关闭
2. 添加管理员备注，记录处理情况
3. 处理完成后可通知用户处理结果

## 六、分析镜头管理
### 6.1 系统镜头管理
1. 查看所有系统内置的分析镜头
2. 修改镜头的提示词、描述等信息
3. 停用/启用系统镜头
4. 发布新版本的分析提示词，支持版本回滚

### 6.2 自定义镜头审核
1. 查看用户创建的自定义镜头
2. 审核违规内容，对包含违法违规内容的镜头进行下架处理
3. 推荐优质自定义镜头为系统镜头

## 七、系统公告管理
1. **发布公告**：创建系统公告，选择公告类型（通知、警告、维护）、填写标题和内容、设置过期时间
2. **公告管理**：编辑、停用、删除已发布的公告
3. 公告会在用户登录时显示在首页顶部

## 八、数据统计
1. **用户统计**：新增用户数、活跃用户数、用户留存率
2. **业务统计**：文档创建量、分析次数、积分消耗量
3. **收入统计**：支付订单数、总金额、套餐销售占比
4. 支持按日、周、月维度查看统计数据

---

## 九、安全注意事项
1. 管理员账号请妥善保管，定期更换密码
2. 所有后台操作都会记录操作日志，请勿进行违规操作
3. 处理用户数据时请遵守隐私保护法规，不得泄露用户个人信息和文档内容
4. 系统升级或维护前请提前发布公告，告知用户维护时间和影响范围
