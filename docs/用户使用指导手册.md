# 碎石记（Crusher）用户使用指导手册

## 一、平台介绍
碎石记是一款AI驱动的多维度文档分析平台，通过不同的"分析镜头"对文本内容进行智能解读，帮助用户快速提取文档核心信息、识别风险、生成专业分析报告。

## 二、账号注册与登录
### 2.1 注册账号
1. 访问平台首页，点击"注册"按钮
2. 填写邮箱地址和密码（密码长度不少于8位）
3. 点击"注册"完成账号创建，系统会自动创建用户资料

### 2.2 登录账号
1. 访问登录页面，填写注册时的邮箱和密码
2. 点击"登录"进入平台主界面
3. 登录成功后，系统会自动跳转至首页

## 三、核心功能使用
### 3.1 文档创建
1. 点击左侧导航栏"创建文档"按钮（PC端）或底部导航栏"+"号（移动端）
2. 输入文档标题（可选）
3. 在文本编辑区粘贴或输入需要分析的文档内容
4. 系统会自动统计字符数，并显示所需的分析积分
5. 点击"保存"完成文档创建

### 3.2 文档分析
1. 在文档详情页，选择需要使用的分析镜头：
   - 📋 需求分析：提取需求点、识别矛盾、评估可行性
   - 📝 会议纪要：整理会议要点、行动项、责任人
   - 🔍 文档评审：识别文档缺陷、提出改进建议
   - ⚠️ 风险评估：识别潜在风险点、给出应对建议
   - 🔄 变更管理：分析变更影响、评估风险
   - 📊 事后复盘：总结经验教训、提出改进方案
   - 🛠️ 技术文档：提取技术要点、整理架构信息
   - ✨ 自定义镜头：用户自行创建的个性化分析模板
2. 确认积分消耗（根据文档字符数自动计算）
3. 点击"开始分析"，等待AI生成分析结果
4. 分析完成后，可查看Markdown格式的分析报告，支持复制和导出

### 3.3 文档管理
1. 点击"我的文档"查看所有已创建的文档
2. 支持按创建时间排序、搜索文档标题
3. 点击文档标题进入详情页，可查看原始文档和所有分析记录
4. 支持删除不需要的文档（删除后不可恢复）

### 3.4 自定义镜头管理
1. 点击"我的镜头"进入镜头管理页面
2. 点击"创建自定义镜头"
3. 填写镜头名称、图标、描述
4. 输入系统提示词（定义AI的分析规则和输出格式）
5. 保存后即可在文档分析时使用自定义镜头

## 四、积分系统
### 4.1 积分计算规则
分析费用根据文档字符数计算：
- ≤3,000字符：10积分
- ≤6,000字符：15积分
- ≤10,000字符：22积分
- >10,000字符：22 + 向上取整((字符数-10000)/1000) * 5

### 4.2 积分购买
1. 点击"个人中心"进入个人资料页面
2. 点击"购买积分"
3. 选择适合的积分套餐：
   - 入门包：100积分/¥10
   - 标准包：500积分/¥45（优惠10%）
   - 专业包：1200积分/¥96（优惠20%）
4. 选择支付方式，完成支付后积分会自动到账

### 4.3 积分查询
1. 在个人中心页面可查看当前积分余额
2. 点击"交易记录"可查看所有积分变动明细，包括购买、消费、退款等记录

## 五、个人中心
1. **资料编辑**：修改用户名、邮箱等个人信息
2. **密码修改**：更新账号登录密码
3. **反馈提交**：提交bug报告、功能建议、支付问题等反馈
4. **退出登录**：安全退出当前账号
