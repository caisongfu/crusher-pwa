## 已知问题汇总

### 1. 管理后台首页客户端错误 🔴
**严重程度**: 高
**影响**: `/admin` 路由无法正常显示
**表现**: 显示 "Application error: a client-side exception has occurred while loading localhost"

**排查步骤**:
1. 在浏览器中访问 `http://localhost:3000/admin`
2. 打开浏览器开发者工具 (F12)
3. 查看 Console 标签页的错误信息
4. 查看 Network 标签页，检查 API 请求是否失败

**可能原因**:
- 客户端组件导入错误
- 依赖组件未正确加载
- API 调用失败导致组件崩溃
- TypeScript 类型错误

**建议修复**:
- 检查 `src/app/admin/page.tsx` 的导入语句
- 验证 `@/components/admin/user-list` 组件是否存在
- 添加错误边界 (Error Boundary)
- 检查 API 调用的错误处理

### 2. 并发登录问题 🟡
**严重程度**: 低
**影响**: 测试并发执行时部分测试失败
**表现**: 多个测试用例同时登录时，部分测试停留在登录页

**可能原因**:
- Playwright 并发测试共享认证状态
- Cookie 或 Session 冲突

**建议修复**:
- 使用 `test.use({ storageState: 'state.json' })` 为每个测试创建独立的浏览器上下文
- 或设置 `workers: 1` 串行执行测试
- 或为每个测试使用独立的浏览器实例
